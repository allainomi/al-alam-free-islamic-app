# Al-Alam Free Islamic App

**بِسْمِ اللّٰهِ الرَّحْمَٰنِ الرَّحِيْمِ**

مفت اسلامی ایپ: قرآن، نماز کے اوقات، ہجری تاریخ، اذکار، حدیث!

## فیچرز
- قرآن پاک (پڑھیں + سنئیں)
- نماز کے اوقات (لوکیشن بیسڈ)
- ہجری/شمسی تاریخ
- اذکار صبح/شام
- حدیث روزانہ

## APK ڈاؤن لوڈ
[Actions → Artifacts → app-release.apk](https://github.com/allainomi/al-alam-free-islamic-app/actions)
